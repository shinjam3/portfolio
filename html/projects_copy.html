Projects

Immersive Environment (2018)

Created a video game world using Unity. Various 3D objects were created using Autodesk Maya, and poster images were created using Adobe Photoshop. Produced a 35 second trailer featuring the world using Adobe Premiere Pro to promote this project.

Jug Puzzle (2017)

Created a variation of the classic 3 jugs puzzle game, implemented with the programming language, Java. There are 3 jugs with the first jug having a maximum capacity of 8 litres, the second jug having a capacity of 5, and the third jug having a capcity of 3. The objective of the puzzle is to have 4 litres of water in both the first and second jug, while having no water in the third jug.

Paint Application (2017)

Created a prototype desktop paint application with a few friends that allows the user to draw on a blank canvas and save their project to a desired location.
